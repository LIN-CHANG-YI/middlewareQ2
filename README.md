每請求一次網站都會顯示
+ 伺服器收到請求的時間戳記
+ 伺服器送出回應的時間
+ 最後算出request-response cycle的時間
# 安裝方式
1.打開 terminal 將專案 clone 到本地電腦
```
git clone https://github.com/LIN-CHANG-YI/middlewareQ2.git
```
2.進入專案資料夾
```
cd middlewareQ2
```
3.安裝npm
```
npm install
```
4.安裝nodemon
```
npm install nodemon
```
5.啟動程式
```
npm run dev
```
6.成功執行
```
在 terminal 可以看到 App running on port 3000
```
7.開啟瀏覽器
```
網址列輸入localhost:3000
```
# 開發環境
+ Node.js: v10.15.0
